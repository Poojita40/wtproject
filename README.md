# Online Book Store Web Application

## Project Overview
- The Online Book Store Web Application is a simple web-based platform that allows users to browse books and manage their shopping cart.
- The system provides pages for login, registration, book catalogue, shopping cart, and user profile.
- This project demonstrates basic web development concepts using HTML, CSS, Bootstrap, and JavaScript.

## Objectives
- To design a simple and interactive online bookstore interface
- To allow users to browse available books
- To provide login and registration functionality
- To allow users to add books to a shopping cart
- To display the total price of selected books

## Technologies Used
- HTML5 for page structure
- CSS3 for styling
- Bootstrap 5 for responsive design
- JavaScript for basic interactivity

## Project Structure
- index.html – Main portal page
- loginpage.html – Login page
- registration.html – User registration page
- catalogue.html – Book catalogue page
- cart.html – Shopping cart page
- profile.html – User profile page
- README.md – Project documentation

## Features

### User Authentication
- Login page for registered users
- Registration page for new users

### Book Catalogue
- Displays available books
- Users can browse books with images
- Option to add books to cart

### Shopping Cart
- Displays selected books
- Shows book price and quantity
- Calculates total price
- Remove items from cart

### Profile Page
- Displays user information
- Shows resume details

### Responsive Design
- Built using Bootstrap
- Works on desktop, tablet, and mobile devices
